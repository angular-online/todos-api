To run the app:
--------------
> nodemon server.js

OR 

> node server.js

Export Entire DB:
-----------------
mongodump -d <database_name> -o <directory_backup>
mongodump -d todos -o dbdump

Import Entire DB:
-----------------
mongorestore -d <database_name> <directory_backup>
dbdump>  mongorestore -d todos todos
